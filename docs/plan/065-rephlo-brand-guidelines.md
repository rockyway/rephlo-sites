## Typography

### Primary Typeface: Inter

```
Font Family: Inter
Weights Available: 300, 400, 500, 600, 700
Source: Google Fonts (Open Source)
License: SIL Open Font License 1.1
Download: https://fonts.google.com/specimen/Inter
```

#### Weight Usage

**Light (300)**: Hero text 48px+
**Regular (400)**: Body text, descriptions, UI labels
**Medium (500)**: Subheadings, button text, navigation
**Semibold (600)**: Section headings, card titles
**Bold (700)**: Hero headings, primary CTAs

#### Type Scale

```
Hero:          48px | Line-height: 1.1  | Weight: 700 | Letter-spacing: -0.02em
Heading 1:     36px | Line-height: 1.2  | Weight: 600 | Letter-spacing: -0.01em
Heading 2:     30px | Line-height: 1.3  | Weight: 600 | Letter-spacing: -0.01em
Heading 3:     24px | Line-height: 1.4  | Weight: 600 | Letter-spacing: 0
Heading 4:     20px | Line-height: 1.4  | Weight: 500 | Letter-spacing: 0
Body Large:    18px | Line-height: 1.6  | Weight: 400 | Letter-spacing: 0
Body:          16px | Line-height: 1.6  | Weight: 400 | Letter-spacing: 0
Body Small:    14px | Line-height: 1.5  | Weight: 400 | Letter-spacing: 0
Caption:       12px | Line-height: 1.4  | Weight: 400 | Letter-spacing: 0.01em
```

#### Letter Spacing

- **Tight** (-0.02em): Large headings (48px+) for better optical balance
- **Normal** (0): Body text and most UI elements
- **Loose** (0.01em): Very small text (captions, labels) for legibility

### Monospace Typeface: JetBrains Mono

```
Font Family: JetBrains Mono
Weights Available: 400, 500, 700
Source: Google Fonts (Open Source)

Usage:
- Code snippets and examples
- Command syntax display
- Technical documentation
- API responses and JSON data
```

### Typography Rules

‚úÖ **DO:**
- Use Inter for all UI and marketing text
- Maintain consistent line-height (1.5-1.6 for body text)
- Use appropriate weights for hierarchy
- Left-align body text (easier to read)
- Use sentence case for headings (more approachable)

‚ùå **DON'T:**
- Mix multiple typefaces (stick to Inter + JetBrains Mono)
- Use all caps for long text (harder to read)
- Set body text below 14px (accessibility)
- Use tight line-height for body text (<1.4)
- Center-align long paragraphs

## Photography & Imagery

### Photography Style

**Characteristics:**
- Clean, modern, professional
- Natural lighting preferred
- Diverse, inclusive representation
- Focus on real people using technology
- Contextual environments (offices, homes, cafes)

**Subjects:**
- People writing, working, creating
- Close-ups of hands typing
- Computer screens showing Rephlo in use
- Modern workspaces and tools

**Avoid:**
- Stock photo clich√©s (forced smiles, pointing at screens)
- Overly staged or artificial setups
- Dark, moody photography
- Cluttered or distracting backgrounds

### Illustration Style: "Flow Lines"

**Visual Language:**
```
Elements:
‚Ä¢ Smooth, flowing curves (like water, air)
‚Ä¢ Abstract geometric shapes
‚Ä¢ Minimal color palette (brand colors only)
‚Ä¢ Subtle gradients on key elements
‚Ä¢ Isometric perspective optional
```

**Use Cases:**
- Empty states ("No commands yet")
- Onboarding step indicators
- Error state explanations
- Feature illustrations
- Marketing hero graphics

**Principles:**
‚úÖ Abstract and metaphorical (not literal)
‚úÖ Simplified shapes and forms
‚úÖ Limited color palette (2-3 colors max)
‚úÖ Purposeful negative space

‚ùå Realistic or photographic illustrations
‚ùå Cluttered or overly detailed
‚ùå Rainbow colors or unrelated palettes
‚ùå Trendy styles that will date quickly

### Iconography

**Recommended Library**: Lucide Icons or Phosphor Icons

**Style Requirements:**
```
Type: Stroke-based (outlined, not filled)
Stroke Weight: 2px
Line Caps: Rounded
Grid: 24x24px
Colors: Brand colors only
```

**Icon Usage:**
- Always pair icons with text labels
- Use consistent size within context (20px for UI, 24px for navigation)
- Maintain visual weight (don't mix outlined and filled)
- Apply brand colors appropriately

## Voice & Tone

### Brand Voice (Consistent Attributes)

**Intelligent but Approachable**
```
‚úÖ "Rephlo uses AI to enhance your writing with context-aware suggestions."
‚ùå "Our advanced neural networks leverage state-of-the-art NLP algorithms."

Explanation: Explain AI concepts clearly without jargon.
```

**Confident yet Humble**
```
‚úÖ "Rephlo helps you write better, faster."
‚ùå "Rephlo revolutionizes writing forever with unmatched AI superiority."

Explanation: Confident in capabilities, humble in claims.
```

**Clear and Direct**
```
‚úÖ "Select text, choose a command, and your content transforms instantly."
‚ùå "Utilizing our innovative workflow methodology, users can harness transformative capabilities."

Explanation: Use simple, active language. Avoid corporate-speak.
```

**Empowering (You-Focused)**
```
‚úÖ "You write with confidence. Rephlo refines with intelligence."
‚ùå "Our AI writes for you so you don't have to."

Explanation: User amplification, not replacement.
```

### Tone by Context

#### Marketing Website (Aspirational + Clear)
```
Headline: "Write anywhere. Enhance everywhere."
Body: "Rephlo brings AI-powered writing assistance to every Windows application. Transform emails, documents, and messages‚Äîright where you're working."
CTA: "Download for Free"
```

#### Product Documentation (Clear + Thorough)
```
Heading: "Creating Your First Command"
Body: "Commands are customizable instructions that tell Rephlo how to transform your text. To create one, click 'Add Command' and enter a name and instruction."
Tone: Instructional, patient, complete
```

#### Error Messages (Helpful + Reassuring)
```
Error: "Couldn't connect to AI service"
Message: "Please check your internet connection and provider settings. If the problem persists, try switching to a different AI provider."
CTA: "Open Settings"
Tone: Non-blaming, solution-focused
```

#### Social Media (Conversational + Smart)
```
Post: "Tired of copy-pasting every email into ChatGPT? üòÖ Same. That's why Rephlo works right where you write‚ÄîOutlook, Slack, anywhere. Download for Windows: [link]"
Tone: Relatable, casual, problem-solving
```

### Writing Guidelines

‚úÖ **DO:**
- Use active voice ("Rephlo transforms text" not "Text is transformed by Rephlo")
- Write in second person ("You can create unlimited commands")
- Use contractions for friendliness ("We're" instead of "We are")
- Keep sentences concise (aim for <25 words)
- Use bullets and lists for scannability

‚ùå **DON'T:**
- Use passive voice excessively
- Write in third person ("The user should...")
- Use formal, stiff language
- Create walls of text without breaks
- Use exclamation marks excessively (1 per page max)

## Common Mistakes

### Logo Mistakes ‚ùå

**Mistake 1: Wrong Colors**
```
‚ùå Using random colors (pink, green, orange logos)
‚úÖ Use only Rephlo Blue, White, or Black
```

**Mistake 2: Distorted Proportions**
```
‚ùå Stretching logo wider or taller
‚úÖ Maintain original aspect ratio, scale proportionally
```

**Mistake 3: Insufficient Clear Space**
```
‚ùå Text or graphics touching logo
‚úÖ Maintain 0.5x logo height clear space on all sides
```

### Color Mistakes ‚ùå

**Mistake 1: Low Contrast Text**
```
‚ùå Cyan (#06B6D4) text on white background (3.9:1)
‚úÖ Deep Navy (#1E293B) text on white (14.8:1)
```

**Mistake 2: Color-Only Communication**
```
‚ùå Red text alone to indicate error
‚úÖ Red text + error icon + descriptive message
```

**Mistake 3: Non-Brand Color Mixing**
```
‚ùå Using purple, lime green, or other non-brand colors
‚úÖ Stick to defined color palette
```

### Typography Mistakes ‚ùå

**Mistake 1: Too Many Fonts**
```
‚ùå Mixing Arial, Times New Roman, Inter, and Comic Sans
‚úÖ Inter for UI/marketing, JetBrains Mono for code only
```

**Mistake 2: Tiny Text**
```
‚ùå 10px body text (accessibility fail)
‚úÖ Minimum 14px for body text, 16px preferred
```

**Mistake 3: Tight Line-Height**
```
‚ùå 1.2 line-height for paragraphs (cramped)
‚úÖ 1.5-1.6 line-height for body text (readable)
```

### Voice & Tone Mistakes ‚ùå

**Mistake 1: Overly Technical**
```
‚ùå "Leverage our advanced LLM API integration for transformative NLP workflows."
‚úÖ "Use AI to enhance your writing with smart suggestions."
```

**Mistake 2: Passive Voice Overload**
```
‚ùå "Your text is transformed by our AI engine."
‚úÖ "Rephlo transforms your text with AI intelligence."
```

**Mistake 3: Excessive Hype**
```
‚ùå "THE MOST REVOLUTIONARY AI TOOL EVER CREATED!!!"
‚úÖ "AI-powered writing enhancement that works everywhere you write."
```
