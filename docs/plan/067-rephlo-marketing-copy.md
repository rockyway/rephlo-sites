### General Questions

**Q: What is Rephlo?**
```
A: Rephlo is an AI-powered writing assistant for Windows that works system-wide across all applications. Instead of copying text to ChatGPT or other tools, Rephlo lets you enhance text directly where you're working‚Äîin emails, documents, code editors, chat apps, and more. Simply select text, choose a command, and see AI-enhanced results instantly.
```

**Q: How much does Rephlo cost?**
```
A: Rephlo is currently free during beta. The free tier includes unlimited custom commands, system-wide text enhancement, and works with both cloud AI providers (100 transformations/month) and unlimited local AI via Ollama. Paid tiers with additional features will be introduced after the beta period.
```

**Q: What's the difference between Rephlo and ChatGPT/Claude?**
```
A: ChatGPT and Claude are chat interfaces where you copy text, paste it into their web app, wait for a response, then copy results back. Rephlo integrates directly into Windows, allowing you to transform text in-place without ever leaving your current application. Think of Rephlo as bringing ChatGPT-level intelligence directly into every app you use, with customizable commands tailored to your needs.
```

**Q: Does Rephlo replace Grammarly?**
```
A: Rephlo and Grammarly serve different purposes. Grammarly focuses on real-time grammar correction and writing suggestions, primarily in browsers. Rephlo is about on-demand AI transformation using large language models‚Äîrewriting, summarizing, tone adjustment, and custom transformations across all Windows applications. Many users use both: Grammarly for passive correction, Rephlo for active transformation.
```

### Privacy & Security Questions

**Q: Is my data secure?**
```
A: Yes. Rephlo takes privacy seriously:

1. Zero Logging: We never log, store, or transmit your text to our servers
2. Local Processing Option: Use Ollama for complete on-device AI processing
3. Encrypted Storage: API keys encrypted with Windows DPAPI (AES-256)
4. No Telemetry: Zero analytics or usage tracking without explicit opt-in
5. Transparent Cloud Providers: When using cloud AI, we clearly indicate which provider processes your text

When text is sent to cloud providers, it's transmitted via HTTPS and subject to their privacy policies (Azure, Groq). We recommend Ollama for sensitive or confidential content.
```

**Q: Does Rephlo send my text to the internet?**
```
A: Only when using cloud AI providers (Azure OpenAI, Groq). When you select a command, the selected text is sent via encrypted HTTPS to the provider you've configured for AI processing, then discarded after receiving the response.

If you use Ollama (local AI), all processing happens entirely on your machine with zero network transmission. This is ideal for sensitive documents, proprietary code, or regulated industries.
```

**Q: Where are my API keys stored?**
```
A: API keys for cloud providers are encrypted and stored locally on your machine using Windows Data Protection API (DPAPI) with AES-256 encryption. Only your Windows user account can decrypt these keys. They are never transmitted to our servers or stored in the cloud.
```

**Q: Can I use Rephlo for confidential/proprietary work?**
```
A: Yes, with Ollama. Configure Rephlo to use local Ollama models, and all AI processing happens entirely on your machine. Your text never leaves your computer, making it suitable for:
‚Ä¢ Confidential business documents
‚Ä¢ Proprietary code and technical documents
‚Ä¢ Regulated industries (healthcare, legal, finance)
‚Ä¢ Government or military work with classification requirements
‚Ä¢ Any sensitive content requiring air-gapped processing
```

### Troubleshooting Questions

**Q: Rephlo context menu doesn't appear when I select text. Why?**
```
A: Common solutions:

1. Ensure Rephlo is running (check system tray for blue icon)
2. Hold Ctrl key while releasing mouse button after text selection
3. Try right-clicking selected text to see if menu appears
4. Check if the application is compatible (95%+ are, but some legacy apps may not work)
5. Restart Rephlo from the system tray menu

If issues persist, contact support@rephlo.io with the application name and Windows version.
```

**Q: My AI transformations are slow. How can I speed them up?**
```
A: Speed depends on your AI provider:

Cloud Providers (Azure, Groq):
‚Ä¢ Check internet connection speed
‚Ä¢ Try switching to Groq (generally faster than Azure)
‚Ä¢ Smaller text selections process faster

Local Ollama:
‚Ä¢ Use smaller models (Phi-3, Mistral 7B) for faster processing
‚Ä¢ Ensure sufficient RAM (8GB minimum, 16GB recommended)
‚Ä¢ Enable GPU acceleration if you have a compatible graphics card

Typical transformation times: 3-10 seconds depending on text length and provider.
```

**Q: I'm getting "API Error" when using cloud providers. What's wrong?**
```
A: Common causes:

1. Invalid API Key: Re-enter your API key in Settings ‚Üí LLM Provider
2. Expired Subscription: Check your Azure/Groq account for active subscription
3. Rate Limits: You may have exceeded provider's rate limits (wait a few minutes)
4. Network Issues: Verify internet connection
5. Provider Outage: Check provider status pages (Azure Status, Groq Status)

Try the "Test Connection" button in Settings to diagnose the issue.
```
